# 🚑 TrackNHeal: Ambulance Booking & Tracking System

## 📘 Project Overview
**TrackNHeal** is a robust web application designed to streamline emergency medical workflows. It provides a structured interface for booking ambulances, managing patient records, and ensuring secure user authentication.

## 🚀 Key Features
- **User Authentication:** Secure login/signup using **Bcrypt** password hashing.
- **Booking Management:** Real-time interface for patient pickup and drop-off coordination.
- **Admin Dashboard:** Centralized control for managing users and emergency bookings.
- **Relational Database:** Optimized **MySQL** schema for data integrity.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Node.js (Express)
- **Database:** MySQL
- **Security:** Bcrypt (10 Salt Rounds)
